# Encontro 7 - Git e GitHub Básico

## Objetivos
- Entender o que é controle de versão
- Aprender comandos básicos do Git
- Publicar projeto no GitHub

## Teoria (40 min)

### O que é Git?
Sistema de controle de versão que registra mudanças no código ao longo do tempo.

### O que é GitHub?
Plataforma online para hospedar repositórios Git e colaborar com outros desenvolvedores.

### Conceitos Principais
- **Repositório:** Pasta do projeto com histórico de versões
- **Commit:** Salvar uma versão do código
- **Push:** Enviar commits para o GitHub
- **Clone:** Baixar um repositório

### Comandos Básicos

```bash
git init
git add .
git commit -m "Mensagem descritiva"
git remote add origin URL
git push -u origin main
```

## Prática (1h20)

1. Instalar Git
2. Configurar nome e email
3. Inicializar repositório no projeto do cardápio
4. Fazer primeiro commit
5. Criar repositório no GitHub
6. Fazer push do projeto

Veja os exemplos na pasta `exemplos/` e complete os desafios na pasta `desafios/`.

## Tarefa de Casa

- Fazer pelo menos 2-3 commits com melhorias
- Criar README.md básico no repositório
- Deixar o repositório organizado
