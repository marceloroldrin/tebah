# Encontro 6 - Bibliotecas CSS

## Objetivos
- Conhecer diferentes bibliotecas CSS
- Entender quando usar bibliotecas
- Testar Normalize.css e componentes de Bootstrap/Tailwind

## Teoria (40 min)

### Tipos de Bibliotecas

**Normalize.css / Reset CSS**
- Normaliza estilos entre navegadores
- Remove inconsistências padrão

**Bootstrap**
- Framework completo com componentes prontos
- Sistema de grid responsivo
- Componentes JS inclusos

**Tailwind CSS**
- Classes utilitárias
- Abordagem utility-first
- Altamente customizável

### Vantagens e Desvantagens

**Bibliotecas:**
- ✅ Rapidez no desenvolvimento
- ✅ Componentes testados
- ❌ Menos controle total
- ❌ Pode ficar genérico

**CSS Puro:**
- ✅ Controle total
- ✅ Personalização completa
- ❌ Mais tempo de desenvolvimento
- ❌ Precisa testar tudo

## Prática (1h20)

1. Adicionar Normalize.css ao projeto
2. Testar componente de Bootstrap (botão ou card)
3. Testar classes do Tailwind
4. Discutir quando usar cada abordagem

Veja os exemplos na pasta `exemplos/` e complete os desafios na pasta `desafios/`.

## Tarefa de Casa

Ajustar o cardápio com Normalize.css e decidir se algum componente usará biblioteca.

## Uso de IA

- Peça exemplo de botão em Bootstrap e Tailwind
- Compare com sua solução manual
- Pergunte sobre prós e contras de cada biblioteca
