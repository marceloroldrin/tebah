# Encontro 8 - Apresentação dos Cardápios (HTML + CSS)

## Objetivos
- Apresentar os projetos completos
- Receber e dar feedback
- Celebrar o progresso

## Teoria (20 min)

### Recapitulação do Mês 2

**O que aprendemos:**
- Fundamentos de CSS e Box Model
- Flexbox para layouts em uma dimensão
- Grid para layouts em duas dimensões
- Responsividade com mobile-first
- Padrão BEM para organização
- Bibliotecas CSS (Normalize, Bootstrap, Tailwind)
- Transições e animações

**Principais conceitos:**
- Box Model: margin, padding, border, content
- Flexbox: display flex, justify-content, align-items
- Grid: grid-template-columns, gap
- Media queries: @media (min-width: ...)
- BEM: block__element--modifier
- Transições: transition, hover
- Animações: @keyframes, animation

## Prática (1h40)

### Apresentações
Cada aluno (ou grupo) apresenta seu cardápio completo:
- Estrutura HTML e CSS
- Decisões de design
- Uso de Flexbox e Grid
- Responsividade
- Animações aplicadas

### Feedback
- **Técnico:** HTML/CSS está correto?
- **Visual:** Design está atrativo?
- **UX:** É fácil de usar?
- **Responsivo:** Funciona em todos os tamanhos?

### Discussão
- Quais desafios tiveram?
- Como a IA ajudou?
- O que foi resolvido manualmente?
- Qual parte foi mais difícil?
- Qual parte foi mais divertida?

## Tarefa de Casa

- Ajustar o projeto com base no feedback
- Fazer commit final das melhorias
- Deixar o cardápio pronto para JavaScript no Mês 3

## Preparação para o Mês 3

No próximo mês vamos:
- Aprender JavaScript
- Adicionar interatividade ao cardápio
- Trabalhar com eventos (cliques, formulários)
- Manipular o DOM
- Criar funcionalidades dinâmicas
